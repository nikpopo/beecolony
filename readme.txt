Параметр количество пчел-собирателей отвечает за то, 
какое количество днф будет использоваться в эпохе.
Использовать можно любое количество, чем меньше, тем быстрее работает, но на маленьких значениях смысла нет.

Параметр количество пчел-разведчиков отвечает за возможности исследования новых
днф, которые изначально не были найдены пчелами-собирателями.
Чем больше, тем быстрее сводится к лучшему реешнию, так как все новые возможные варианты слишком быстро иссекают.
Лучше использовать значения до 1000

Количество итераций следует повышать, если видно, что среднее не сошлось с лучшим, так есть возможность найти новые интересные варианты.
